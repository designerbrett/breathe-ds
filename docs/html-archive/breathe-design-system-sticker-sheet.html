Typography

Optimized for readability across all age groups. Larger base sizes for mobile (65+) audience. Lexend was designed for accessibility (originally for dyslexic readers), making it perfect for our 65+ audience while maintaining brand consistency with kivohealth.com.

Accessibility Note: Base font size is 17px (mobile) / 16px (desktop) for improved readability. Line height minimum 1.5 for body text. WCAG AAA contrast ratios maintained throughout.

Component Usage Guidelines

Button Hierarchy (Inspired by Function & Hinge Health)
Primary (Filled Blue): Main action, forward progress, completion, submission. The most important action on the page.
Secondary (Outlined Blue): Alternative actions, less critical paths, secondary workflows. Uses same brand color but with less visual weight.
Tertiary (Text Link Blue): Low-priority actions, back navigation, inline links, dismissals. Text-only, no background or border.
Destructive (Red): Only for actual destructive actions like delete, remove, cancel subscription. Not for general secondary actions.
Card elevation hierarchy
Standard: Default for most content grouping
Elevated: Interactive cards, clickable content, important callouts
Bordered: Dense layouts, nested cards, subtle grouping
Color Application (Minimal Approach)
Kivo Blue (#2B79B9): Your primary brand color. Use for all primary actions, links, focus states, interactive elements, and brand moments. This creates consistency and trust.

Neutrals (Grays): Use for text hierarchy, backgrounds, borders, and non-interactive UI. Most of your interface should be neutral.

Status Colors: Green (success), Orange (warning), Red (error/destructive), Blue (info). Only use for actual status communication.

Accent Colors (Use Sparingly):
‚Ä¢ Gold (#F4B942): Achievements, milestones, celebrations only
‚Ä¢ Peach (#FFB088): Subtle highlights in data visualizations

Following Function Health and Hinge Health's minimal approach: one strong brand color + neutrals + functional status colors.
